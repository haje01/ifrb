.. _object:

객체(Object)
============

객체는 IF내의 모든 것의 :ref:`super` 이다. 객체가 가지는 특성은 이 후에 설명할
물체, 캐릭터, 공간 등 모든 것에 공통적으로 적용된다. 

*다음과 같은 속성이 있다.*

.. _id:

아이디(ID)
----------
:ref:`scenario` 안에서 각 객체가 가지는 유니크한 ID, 중복되어서는 안된다.

:ref:`type-id` 이다.

.. _name:

이름(Name)
----------
:ref:`play` 시 객체가 불려지는 명칭을 말한다. 서로 다른 객체가 같은 이름을 가질
수 있으나, 유저가 혼란스럽지 않도록 하자.

:ref:`type-string` 이다.
