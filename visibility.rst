가시도(Visibility)
==================

가시도는 :ref:`character` 가 특정 :ref:`object` 를 볼 수 있는 지를 나타낸다. 다음과 같이 결정된다.

#. 캐릭터가 위치한 :ref:`space` 의 :ref:`luminance` 가 깜깜하면 보이지 않는다.
#. 캐릭터와 대상 객체 사이에 불투명한 공간이 있으면 보이지 않는다.
#. 이외의 경우 보인다.

.. note::
   가시도의 대상은 물체가 아닌 객체인 것에 주의하자. 가시도는 공간에 대해서도 적용된다.
